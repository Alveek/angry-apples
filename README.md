# 🐍 Snake Game (Phaser.js)

Классическая игра Snake с уникальными механиками, построенная на **Phaser 3**.

## 🎮 Возможности

- **Движущаяся добыча** — яблоко не стоит на месте, а пытается убежать от змеи
- **Нитро-ускорение** — зажмите Space, чтобы ускориться (с индикатором в UI)
- **Эволюция добычи** — каждые 5 съеденных яблок вместо обычной добычи появляется **Охотник**
- **Охотник** — фиолетово-красная добыча, которая гоняется за хвостом змеи со скоростью 130мс (быстрее обычной змеи)
- **Стратегия** — змея может съесть Охотника головой (+50 очков), но Охотник может откушать хвост (-10 очков)
- **Дрожание при застревании** — яблоко задрожит, если не может найти путь
- **Весёлые зефир** — редкое случайное ускорение добычи на 1–2 секунды
- **Пауза** — Enter для паузы/рестарта
- **Пиксель-арт графика** — вся графика генерируется программно через Phaser Graphics

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| ↑↓←→ / WASD | Движение змеи |
| Space | Нитро-ускорение (с ограниченным ресурсом) |
| Enter | Пауза / Рестарт при Game Over |

## 🚀 Запуск

### Локально (HTTP-сервер)

```bash
python3 -m http.server 8080 --directory .
```

Откройте `http://localhost:8080` в браузере.

### Тесты

```bash
# Unit-тесты (Node.js)
node test.js

# Браузерные тесты (Playwright, Chrome)
node test-browser.js

# Все тесты
npm test
```

## 🤖 AI Добычи

**Обычная добыча:**
1. Вычисляет вектор от змеи (убегание)
2. Двигается в направлении наибольшего расстояния
3. Запоминает предыдущую позицию — не может вернуться обратно (антибликинг)
4. При застревании 3+ тиков — дрожит (удалено в пользу стратегического геймплея)

**Охотник (Hunter):**
1. Появляется каждую 5-ю добычу
2. Целится в хвост змеи
3. Проходит сквозь тело, но не через голову
4. При контакте с хвостом — удаляет сегмент (-10 очков)
5. Уничтожается при контакте с головой (+50 очков)
